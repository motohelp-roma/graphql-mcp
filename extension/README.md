# Motohelp GraphQL - Claude Desktop Extension

Questa estensione permette a Claude di interagire con le API GraphQL di Motohelp.

## Installazione

1. Scarica `motohelp-graphql.mcpb`
2. Doppio click sul file
3. Claude Desktop si apre - clicca "Install"

## Tools disponibili

- **introspect**: Esplora lo schema GraphQL
- **execute**: Esegui query o mutation GraphQL

## Requisiti

- Claude Desktop >= 1.0.0
- Connessione internet
- Server MCP attivo su https://api.dev.officinemoto.com/mcp

## Build

```bash
chmod +x build.sh
./build.sh
```
